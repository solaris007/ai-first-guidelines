## Viewing the Presentation

The presentation uses **Mermaid diagrams**. To view properly:

- **VS Code**: Install "Markdown Preview Mermaid Support" extension
- **GitHub**: Renders mermaid natively
- **Marp**: Export to PDF/PPTX with `marp evaluation_presentation.md`
