# AI-First Development Guidelines

Comprehensive guidelines for teams adopting AI-first development practices with Claude Code, Codex, Cursor, Gemini CLI, and Copilot.

## What This Repo Contains

Documentation organized in numbered sections covering the full AI-first development lifecycle - from philosophy to implementation to leadership guidance.

## Quick Start

1. **New to AI-first?** Start with [Philosophy](docs/01-foundations/philosophy.md)
2. **Setting up your environment?** See [Workspace Setup](docs/01-foundations/workspace-setup.md)
3. **Ready to build?** Follow the [Development Lifecycle](docs/02-lifecycle/overview.md)
4. **Configuring AI tools?** See [Configuration](docs/04-configuration/)

## Conventions

- All content is markdown
- MUST/MUST NOT for non-negotiable rules, SHOULD/SHOULD NOT for recommendations
- Every concept includes a practical example
- Cross-references use relative paths
- PRs required for all changes
